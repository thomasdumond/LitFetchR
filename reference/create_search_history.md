# Creates an excel file to store the history of searches made using `create_save_search()`

Creates an excel file to store the history of searches made using
[`create_save_search()`](https://thomasdumond.github.io/LitFetchR/reference/create_save_search.md)

## Usage

``` r
create_search_history()
```

## Value

"history_search.xlsx" and or a new sheet for the new search(es)
