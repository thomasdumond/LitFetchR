# create and save search string

`create_save_search` is an interactive function that ask the user to
enter a search string and provide the number of results from 3
platforms: Web of Science, Scopus and PubMed. You can then save one or
more search strings to retrieve the references later.

## Usage

``` r
create_save_search()
```

## Value

one or more search string(s) in the file "search_list.txt"
