# deduplicate the references from three dataframes

deduplicate the references from three dataframes

## Usage

``` r
dedup_refs(df1, df2, df3)
```

## Arguments

- df1:

  dataframe 1

- df2:

  dataframe 2

- df3:

  dataframe 3

## Value

A CSV file containing all the new references deduplicated and the
history of the deduplication.
